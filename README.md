# Online Signature

Simple one-page website to annotate a PDF with a signature image and optional
text. Everything is done client-side.

I (vibe-coded) this because I often needed to "sign" PDF sent to me.

![Screenshot of the app](https://projects.pixelastic.com/signature/screenshot.png)

## Usage

Go to https://projects.pixelastic.com/signature/ for the live version.

Or run `yarn run serve` to run it locally.
